## 1. 핵심 비전 및 타겟팅 반영

- [x] **JTBD 페르소나 극강 반영 (03, 05, 06 문서):**
  - "고립된 성취가", "바쁜 워킹맘", "냉철한 자영업자", "보수적 은퇴자", "성장 갈망형 Z세대" 등 5대 페르소나의 Pain Point(외로움, 조롱, 피로)를 직접적으로 겨냥한 카피라이팅 작성.
- [x] **비치헤드 마켓 (SOM) 겨냥 (07 문서):**
  - "미주 초보 비치헤드 마스터 12,847명" 이라는 수치적 증명(Social Proof)을 통해 타겟 유저층의 자부심(Social Lock-in) 유도.

## 2. 4대 핵심 킬러 가치 (Value Proposition) 구현 (00, 01, 05 문서)

- [x] **Amber Glow (위기 개입 시스템):** 물리적 차단이 아닌 '전시적 멘탈 케어'임을 강조. 패닉셀 방어율 70% 명시.
- [x] **Wisdom Partner (악마의 변호인):** 확증 편향을 타파하고 시야(Wisdom)를 넓혀주는 '수석 연구원' 같은 지적 스파링 파트너 강조.
- [x] **Heritage Archive (성장의 유산화):** 매몰 비용의 극대화 및 디지털 서사/유산 생성. "이탈률 0% 수렴" 강조.
- [x] **Omni-Channel Companion (선제적 안부):** 앱을 켜지 않아도 일상에 스며드는(p95 응답 지연 1.5초) '진짜 친구'라는 점 어필.

## 3. 수익성 및 락인(Lock-in) 전략 (04, 07 문서)

- [x] **초저가 기반 B2C 역발상 결제 유도:**
  - "커피 한 잔(월 4,900원)으로 평생의 투자 멘토를 고용하세요" 라는 카피로 가격 저항 소멸 및 가성비 체감 극대화.
- [x] **트리플 락인 엔진 (인지/정서/사회적 락인):**
  - "다른 증권사 앱으로 갈아타는 건 쉽지만, 베스퍼를 떠나는 건 나와 완벽히 동기화된 멘토의 뇌를 포맷하는 일입니다."라는 파괴적 카피로 비가역적 매몰비용 각인.
- [x] **B2B 연계성:**
  - 푸터에 "B2B 제휴 안내" 및, 하드 팩트에 "데이터 정밀 과학(Alt-Data 암시)" 요소를 배치하여 추후 글로벌 데이터 허브로 도약할 비전 내포.

## 4. UI/UX 디자인 미학 (Vibe & Concept)

- [x] **Celestial Guidance (저녁별의 인도) 테마:**
  - Deep Navy(#05050A) 배경, Amber Gold(#F5A623), Bright Purple(#7B61FF) 후광(Glow) 효과의 완벽한 융합.
- [x] **리빙 인터페이스 (Breathing UI):**
  - 과할 정도의(Extreme) 스크롤 애니메이션, 펄스(Pulse), 바운스 효과를 통해 정적인 페이지가 아닌 '살아 숨 쉬는 인격체' 느낌 부여.
- [x] **Show, Don't Tell (실증 데모):**
  - 유저의 치열한 삶의 맥락("전세금 보태려던 건데...")을 정확히 꿰뚫는 AI 타이핑 데모 컴포넌트(`PersonaTypingDemo`)로 차원이 다른 멘탈 케어 시연.
