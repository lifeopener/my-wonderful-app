## STEP 1. [공통] 핵심 코어 필수요소 점검

### ① 히어로 섹션 (Hero Section) - 첫인상 3초

- [x] **명확한 헤드라인:** `"투자의 외로움, 이제 베스퍼가 함께합니다"` — 기능이 아닌 고객의 최종 이득(외로움 해소, 동반자 획득)을 한 문장으로 전달.
- [x] **보조 설명 (Sub-head):** `"당신의 투자 성향을 학습하고, 위기의 순간엔 이성을 지켜주며, 매일 실질적인 인사이트로 꿈의 실현을 함께 이끌어가는 AI 투자 동반자."` — 구체적인 방법(학습, 위기 개입, 인사이트)을 명시.
- [x] **직관적 이미지:** AI 생성 히어로 목업 이미지(앱 화면 + 포트폴리오 차트 + 채팅 UI)를 배치. 글로우 이펙트로 프리미엄 느낌 강화.

### ② 행동 유도 (CTA) - 전환 장치

- [x] **동사형 버튼:** `"지금 무료로 시작하기"` — 명확한 동사형 행동 유도.
- [x] **반복 배치:** CTA 버튼이 **3곳**에 반복 노출됨:
  1. **상단:** 네비게이션 바 `"앱 시작하기"` + 히어로 섹션 CTA 버튼
  2. **중간:** 히어로 섹션 내 `"데모 보기 ↓"` 보조 CTA
  3. **하단:** 최종 CTA 섹션 `"지금 무료로 시작하기"`

### ③ 신뢰의 증거 (Social Proof) - 의심 제거

- [x] **숫자/로고 증명:** 3가지 수치 지표를 카운트업 애니메이션과 함께 제시:
  - `12,847+` 사전 등록 사용자
  - `97%` 만족도
  - `384K+` 누적 AI 대화 세션

## STEP 2. [선택] 유형별 전략 요소 점검

### 🅱️ B 유형 (기술 몰입형) — 주 전략

- [x] **[데모 시연]** `TypingDemo` 컴포넌트로 실시간 타이핑 애니메이션 구현. 사용자 질문 → 타이핑 인디케이터 → AI 응답이 순차적으로 재현되어 "실제 사용 경험"을 시뮬레이션.
- [x] **[하드 팩트]** 4가지 정확한 스펙 수치를 카드 UI로 제시:
  - `≤ 1.5초` 옴니채널 응답 속도 (p95)
  - `95%+` RAG 맥락 기억 정확도
  - `≤ 0.5초` 위기 개입 지연시간
  - `99.5%` 메시지 발송 성공률
- [x] **[전문가 UI]** 전체 랜딩페이지가 다크모드(`#0A0A12` 배경) 기반으로 설계. 앰버/골드(`#F5A623`) 포인트 컬러로 프리미엄 핀테크 무드 구현.

### 🅰️ A 유형 (불안 해소형) — 보조 전략

- [x] **[워크플로우]** 4단계 올인원 투자 여정 도식화: `페르소나 설정 → 실시간 모니터링 → AI 코칭 → 위기 방어`. 화살표 연결로 흐름 시각화.
- [x] **[안전 장치]** 3가지 안전 요소 명시:
  - `🔒 금융 특화 데이터 학습` — AI 환각 방지
  - `🧪 Fallback 안전망` — 장애 시 즉시 안전 템플릿 전환
  - `📡 옴니채널 보호` — 선제적 위기 알림
- [x] **[권위자 추천]** *현재 미구현* — 향후 베타 테스터 추천사 또는 금융 전문가 인용 추가 예정.

> ⚠️ **권위자 추천 항목은 실제 추천사 확보 후 업데이트 필요**

**유형별 전략 점검 결과: 5/6 (권위자 추천 1건 보류)**

## STEP 3. [최종] 자가 피드백

### Q. 고객이 우리 랜딩페이지를 보고 가질 첫 번째 생각은?

> **"투자할 때 늘 외로웠는데, 이런 AI 파트너가 있다면 써보고 싶다"**

> - ✅ 히어로 헤드라인이 고객의 핵심 Pain("투자의 외로움")을 정면으로 겨냥
> - ✅ 타이핑 데모가 AI의 실제 대화 품질을 즉각적으로 체감하게 함
> - ✅ 하드 팩트 수치가 "진짜 되는 거구나"라는 신뢰감 부여

### Q. 경쟁사 대비 우리 페이지만의 '한 방(Kick)'은?

- [ ] `압도적인 권위(A)`
- [x] **`놀라운 시연 영상(B)`** ← 실시간 타이핑 데모 + 패닉셀 방지율 70% 수치

> **핵심 Kick:** 단순한 "AI 챗봇"이 아니라, **위기 시 매매 버튼을 잠그고 감정을 컨트롤해주는 유일무이한 AI 파트너**라는 포지셔닝. 이 독보적인 기능을 타이핑 데모와 하드 팩트로 입증.
