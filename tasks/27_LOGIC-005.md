# [LOGIC-005] `pgvector` 코사인 유사도 검색 (RAG 검색엔진)

## 1. Task 개요
- **도메인:** Backend / Edge Function
- **목표:** 유저의 현재 질문 벡터와 가장 의미가 비슷한 과거 대화 기록(환희, 공포 등)을 DB(`heritage_logs`)에서 유사도 기반으로 뽑아옵니다.
- **선행 태스크:** DB-002, DB-004, LOGIC-004
- **핵심 가치 (VP):** [VP-2] 초밀착 인지 동행 (과거의 경험을 바탕으로 공감하고 조언)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
AI가 유저의 마음을 읽기 위한 핵심 심리 분석 엔진이다. 오늘 질문과 수학적 좌표가 가장 가까운 과거의 기억 조각들을 찾아와야 한다.

작업 지시:
1. Supabase SQL Editor에 접속하여, 벡터 간 코사인 거리(Cosine Distance, `<=>`)를 계산해 유사한 문서를 반환하는 `match_heritage_logs`라는 Postgres Function(RPC) 쿼리를 작성하고 배포하라.
2. `chat/index.ts`에서 유저 질문의 임베딩을 구한 직후, `supabase.rpc('match_heritage_logs', { query_embedding: vector, match_threshold: 0.78, match_count: 5 })`를 호출하라.
3. 검색되어 나온 상위 5개의 과거 대화 내역 텍스트를 하나의 문자열 컨텍스트(Context)로 이어 붙여라.
