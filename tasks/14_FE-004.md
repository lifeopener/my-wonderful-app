# [FE-004] 공통 컴포넌트: `ChatMessageBubble` (Markdown 지원) 작성

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** 유저와 AI의 대화가 표시될 말풍선 UI를 만듭니다. AI가 코드를 주거나 리스트를 줄 때 예쁘게 보이도록 마크다운(Markdown) 파싱을 지원해야 합니다.
- **선행 태스크:** FE-001
- **핵심 가치 (VP):** 가독성 높은 대화창을 통한 옴니채널 일상 교감 극대화

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저는 이 말풍선 화면을 앱에서 가장 오래 보게 될 것이다.
AI의 대답은 단순 텍스트가 아니라 리스트, 볼드체, 심지어 코드 블록도 포함될 수 있다.

작업 지시:
1. `components/ChatMessageBubble.tsx`를 생성하라.
2. props로 `role`('user' | 'assistant')과 `content`(string)를 받아라.
3. `react-native-markdown-display` 패키지를 설치하고 연동하여, `content` 텍스트를 마크다운으로 렌더링하라.
4. NativeWind를 사용해 `role === 'user'`일 때는 말풍선을 우측에, 짙은 색 배경으로 배치하라.
5. `role === 'assistant'`일 때는 말풍선을 좌측에, 부드러운 회색/흰색 배경으로 배치하라.
6. 컴포넌트가 화면 길이를 넘어갈 때 레이아웃이 깨지지 않는지 점검하라.
