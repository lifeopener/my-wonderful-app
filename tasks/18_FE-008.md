# [FE-008] ChatScreen: 메인 대화창 레이아웃

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** AI와 실제로 채팅을 주고받는 핵심 화면의 뼈대(입력창, 전송 버튼, 스크롤 영역)를 만듭니다.
- **선행 태스크:** FE-004
- **핵심 가치 (VP):** 억압적 UI(화면 가림, 입력 막힘) 없이 가장 쾌적하고 자유로운 대화 환경 제공

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저가 가장 오래 머무르는 공간이다. 절대로 타이머를 걸어 입력을 막거나 화면을 블러 처리하는 등의 '억압적 통제' 로직을 넣지 마라. 자유로운 소통이 핵심이다.

작업 지시:
1. `screens/ChatScreen.tsx`를 생성하라.
2. 화면 레이아웃을 크게 2영역으로 나누어라:
   - 상단: 대화 내용이 스크롤되는 `ScrollView` 또는 `FlatList`. 이 안에 미리 만든 `ChatMessageBubble`들을 렌더링하라.
   - 하단: 유저가 텍스트를 칠 수 있는 `TextInput`과 전송 아이콘(Button).
3. 하단 입력창은 여러 줄 입력이 가능하도록 `multiline={true}`로 설정하고, 글자가 길어지면 입력창 높이가 자연스럽게 늘어나도록 만들어라.
---

- [ ] 하단 텍스트 입력창에 글을 길게 쳤을 때 입력창 높이가 늘어나며 레이아웃이 깨지지 않는가?
