# [FE-006] HomeScreen 대시보드 UI

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** 앱 실행 시 가장 먼저 만나는 홈 화면(대시보드)을 만듭니다.
- **선행 태스크:** FE-002
- **핵심 가치 (VP):** 유저의 투자 상태와 AI와의 교감 상태를 시각화하여 지속적인 앱 방문 유도

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저가 앱을 켰을 때, 단순한 챗봇 화면이 아니라 '나를 기다리고 있던 파트너'의 느낌을 줘야 한다.

작업 지시:
1. `screens/HomeScreen.tsx`를 작성하라.
2. 화면 상단에 유저를 환영하는 인사말과, 현재 설정된 AI의 이름(페르소나)을 부드럽게 띄워라.
3. 화면 중앙에는 '오늘의 주요 경제 지표'나 '데일리 투자 미션' 같은 플레이스홀더 카드를 NativeWind를 사용해 우아하게 디자인하라.
4. 화면 하단이나 중앙에 직관적인 둥근 버튼을 만들어, 누르면 `ChatScreen`으로 넘어갈 수 있도록 하라.
5. 디자인은 다크 모드(또는 고급스러운 네이비/블랙 계열)를 베이스로 하고, 포인트 컬러를 활용하라.
---
