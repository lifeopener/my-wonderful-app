**[Prompt]**
너는 지금부터 Vesper AI의 수석 인프라 엔지니어다. 
우리의 철학은 **"고정 서버비 0원(Zero)"**을 유지하는 것이다. 이를 위해 커스텀 백엔드 없이 완벽한 BaaS(Supabase) 환경을 세팅해야 한다.

작업 지시:
1. 현재 디렉토리에서 Supabase CLI를 통해 프로젝트를 초기화(`supabase init`)하라.
2. `package.json`을 점검하고, 로컬 Supabase 환경을 실행할 수 있는 `npm run supabase:start` 등의 스크립트를 추가하라.
3. 생성된 `supabase/config.toml`을 확인하여, 우리가 필요 없는 불필요한 서비스(예: Storage 등 사용 안함)가 낭비되지 않도록 기본 설정을 검토하라.
4. 작업이 완료되면 다음 태스크(DB-002)인 pgvector를 켤 준비가 되었는지 나에게 보고하라.
