# [AI-003] RAG 컨텍스트 및 시스템 룰 조립 (Prompt Builder)

## 1. Task 개요
- **도메인:** Prompt Engineering
- **목표:** AI의 답변 품질을 높이기 위해 RAG 검색 결과, 유저의 페르소나 설정, 그리고 "억압적 UI 금지" 등의 절대 원칙을 하나로 조립하여 프롬프트를 완성합니다.
- **선행 태스크:** LOGIC-005, DB-003
- **핵심 가치 (VP):** [VP-4] 초밀착 인지 동행 (개인화된 대화와 절대적 자유 제공)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
AI에게 정체성과 기억을 주입해야 한다.

작업 지시:
1. `chat/promptBuilder.ts` 함수를 작성하라.
2. 다음 요소들을 하나의 System Message로 합쳐라:
   - `Persona`: 유저가 설정한 이름과 말투 (`profiles` 테이블 데이터)
   - `Heritage`: RAG를 통해 찾은 과거 대화 맥락 (`LOGIC-005` 결과)
   - `Rule Zero`: "절대로 유저를 억압하거나 타이머/블러로 가리지 마라"는 강력한 지침
   - `Compliance`: "불법 투자 자문을 하지 마라"는 가이드라인
3. 이 완성된 프롬프트를 LLM 호출 시 첫 번째 메시지로 전달하라.
---

- [ ] 과거 대화 내용을 언급하며 문맥에 맞는 답변을 하는가?
