# [FE-009] ChatScreen: 키보드 회피 및 자동 스크롤 (UX)

## 1. Task 개요
- **도메인:** Frontend / UX
- **목표:** 모바일 기기 특성상 키보드가 올라올 때 입력창을 가리지 않게 하고, 새 메시지가 오면 자동으로 맨 아래로 스크롤되도록 UX를 고도화합니다.
- **선행 태스크:** FE-008
- **핵심 가치 (VP):** 완벽한 모바일 사용성으로 몰입감(초밀착 인지 동행) 유지

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
키보드가 채팅창을 가리거나, 새 메시지가 왔는데 스크롤을 직접 내려야 하는 불편함은 유저의 몰입감을 철저하게 깨부순다. 완벽한 네이티브 앱 경험을 구현하라.

작업 지시:
1. `ChatScreen.tsx`의 루트 컨테이너를 `KeyboardAvoidingView`로 감싸라. (iOS는 `behavior="padding"`, Android는 `behavior="height"` 등 적절히 분기 처리)
2. 대화 목록을 보여주는 스크롤 컨테이너에 `useRef`를 연결하라.
3. 메시지 배열(데이터)이 업데이트될 때마다 자동으로 스크롤을 맨 아래로 내리는 `scrollToEnd({ animated: true })` 로직을 추가하라.
4. 키보드가 올라올 때 뷰가 부드럽게 밀려 올라가는지 확인하라.
---

- [ ] 새 더미 메시지를 전송했을 때 스크롤이 자동으로 가장 아래로 내려가는가?
