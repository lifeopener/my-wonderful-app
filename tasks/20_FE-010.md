# [FE-010] `useChat` (Vercel AI SDK) React Hook 연동

## 1. Task 개요
- **도메인:** Frontend / Logic
- **목표:** 화면의 채팅 UI와 백엔드 AI 엔진(Edge Function)을 연결하여 실제로 스트리밍 대화를 주고받습니다.
- **선행 태스크:** FE-008
- **핵심 가치 (VP):** 실시간 스트리밍 답변을 통해 딜레이 없이 살아있는 생명체와 대화하는 듯한 경험 제공

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저는 기다림을 싫어한다. Vercel AI SDK를 사용하여 타이핑하듯 한 글자씩 실시간으로 화면에 찍히는 스트리밍 UX를 구현해야 한다.

작업 지시:
1. `@ai-sdk/react` 패키지를 설치하라.
2. `ChatScreen.tsx`에서 `useChat` 훅을 임포트하라. (`const { messages, input, handleInputChange, handleSubmit } = useChat({ api: '여기에 Supabase 엣지 펑션 URL' })`)
3. 기존에 만들어둔 텍스트 입력창과 전송 버튼에 `input`, `handleInputChange`, `handleSubmit`을 연결하라.
4. `messages` 배열을 순회하면서 `ChatMessageBubble`에 데이터를 매핑하라.
5. 아직 서버 함수가 없으므로 API 호출 시 404가 나겠지만, 프론트엔드 상태 관리가 이 훅으로 완전히 대체되었는지 확인하라.
---
