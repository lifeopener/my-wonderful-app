# [DB-009] Auth 가입 시 `profiles` 자동 생성 트리거 작성

## 1. Task 개요
- **도메인:** Database
- **목표:** 사용자가 Supabase Auth로 회원가입을 완료하는 즉시 `public.profiles` 테이블에 해당 유저의 빈 레코드가 자동으로 생성되도록 데이터베이스 트리거를 설정합니다.
- **선행 태스크:** DB-003
- **핵심 가치 (VP):** 끊김 없는 사용자 경험 (가입 즉시 페르소나 설정 가능)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저가 가입한 후 수동으로 프로필을 만들게 시키면 이탈이 발생한다. 가입 즉시 DB가 알아서 프로필 자리를 만들어줘야 한다.

작업 지시:
1. `auth.users` 테이블에 새 레코드가 삽입될 때 실행될 `handle_new_user` 함수를 PostgreSQL로 작성하라.
2. 해당 함수는 `new.id`를 받아 `public.profiles` 테이블에 `id`를 넣고 나머지는 기본값으로 채워야 한다.
3. 이 함수를 호출하는 `on_auth_user_created` 트리거를 `auth.users` 테이블에 생성하라.
4. 모든 로직을 마이그레이션 파일로 만들어 배포하라.
