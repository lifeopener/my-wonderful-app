# [FE-LOGIC-003] 네트워크 에러 시 Fallback 로컬 메시지 렌더링

## 1. Task 개요
- **도메인:** Frontend / Stability
- **목표:** 인터넷이 끊기거나 서버가 응답하지 않을 때, 유저에게 에러 메시지를 대화창 내에 자연스럽게 띄워 불안감을 해소합니다.
- **선행 태스크:** FE-LOGIC-001
- **핵심 가치 (VP):** [CP-2] 비상 상황에서도 앱이 죽지 않는 안정성 확보

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
서버가 죽었다고 해서 앱이 하얀 화면으로 멈추면 유저는 떠난다. 채팅창 안에서 상황을 설명하고 다시 시도를 유도하라.

작업 지시:
1. `useChat` 훅의 `onError` 콜백을 구현하라.
2. 에러 발생 시, 현재 메시지 목록에 `role: 'assistant'`, `content: '죄송합니다. 서버와 연결이 잠시 끊겼습니다.'` 와 같은 가상 메시지를 임시로 추가하라.
3. 해당 에러 메시지 하단에 '다시 보내기' 버튼을 작게 배치하여 유저가 재시도할 수 있게 하라.
---

- [ ] 재시도 버튼 클릭 시 실패했던 메시지가 다시 전송되는가?
