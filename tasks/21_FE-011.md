# [FE-011] 대화 중 페르소나 실시간 스위칭 (UI/Logic)

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** 채팅 중 유저가 기분에 따라 '다정한 친구'에서 '냉철한 멘토' 등으로 AI 페르소나를 즉시 전환할 수 있는 드롭다운 UI를 만듭니다.
- **선행 태스크:** FE-008, FE-010
- **핵심 가치 (VP):** [VP-1] 무한 페르소나와 일상 교감

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저는 매일 감정이 바뀐다. 오늘은 위로가 필요하고, 내일은 팩트 폭행이 필요할 수 있다. 언제든 AI의 영혼을 갈아끼울 수 있는 UI가 필요하다.

작업 지시:
1. `ChatScreen.tsx`의 상단 헤더 부분에 `PersonaSwitcher.tsx` 컴포넌트를 부착하라.
2. 유저가 생성해 둔 페르소나 목록을 보여주는 드롭다운(또는 모달)을 구현하라.
3. 드롭다운에서 페르소나를 변경하면, 현재 `useChat` 훅 또는 전역 상태(Zustand 등)에 선택된 페르소나 ID와 톤앤매너(Tone)가 업데이트되도록 로직을 작성하라.
4. 변경 즉시 다음 채팅부터는 새 페르소나의 성격대로 대답이 오는지 테스트하라.
