# [INFRA-001] Edge Function 배포 및 Secret Keys 주입 세팅

## 1. Task 개요
- **도메인:** Infra / Deployment
- **목표:** 로컬에서 개발한 엣지 함수들을 Supabase 운영 환경에 배포하고, OpenAI 및 Tavily API 키 등의 민감 정보를 안전하게 주입합니다.
- **선행 태스크:** AI-001, LOGIC-001
- **핵심 가치 (VP):** 실제 운영 환경으로의 안전한 런칭

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
로컬 테스트는 끝났다. 이제 전 세계 유저가 쓸 수 있게 클라우드로 배포하라. 키 관리는 철저해야 한다.

작업 지시:
1. `supabase functions deploy chat` 및 `cron-push` 명령어를 실행하라.
2. `supabase secrets set OPENAI_API_KEY=... TAVILY_API_KEY=...` 명령어를 통해 환경변수를 운영 서버에 주입하라.
3. 배포된 URL을 확인하고, 앱 프론트엔드의 환경변수(`EXPO_PUBLIC_API_URL`)를 운영 URL로 교체하라.
---
