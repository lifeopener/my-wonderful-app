# [FE-LOGIC-002] SSE 스트리밍 데이터 실시간 렌더링 최적화

## 1. Task 개요
- **도메인:** Frontend / Rendering
- **목표:** 서버에서 내려오는 SSE(Server-Sent Events) 스트리밍 데이터를 끊김 없이 부드럽게 화면에 그립니다.
- **선행 태스크:** FE-LOGIC-001
- **핵심 가치 (VP):** 실제 사람(또는 생명체)과 대화하는 듯한 매끄러운 타이핑 효과 구현

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
텍스트가 한꺼번에 툭 나타나는 게 아니라, 타이핑되듯이 부드럽게 나타나야 유저가 살아있는 지능과 대화한다고 느낀다.

작업 지시:
1. `ChatMessageBubble.tsx` 컴포넌트 내에서 텍스트가 바뀔 때 불필요한 전체 리렌더링이 일어나지 않도록 최적화하라. (`memo` 사용 검토)
2. 스트리밍 중인 메시지의 경우, 마지막에 깜빡이는 커서(Cursor) 효과를 주어 '생성 중'임을 시각적으로 나타내라.
3. 텍스트가 추가될 때마다 레이아웃이 덜덜 떨리지 않도록 높이 값을 안정적으로 관리하라.
