**[Prompt]**
유저와 AI의 양방향 대화는 모두 1536차원 벡터로 변환되어 저장되는 핵심 자산이다. 빈 메시지나 악의적인 스크립트가 백엔드로 넘어오지 않도록 막아내야 한다.

3. 추출된 TypeScript 타입을 export 하라.
