# [FE-007] PersonaSetupScreen: 페르소나 셋업 폼

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** 유저가 처음 앱에 진입했을 때, AI의 이름과 말투(성격)를 설정하는 화면을 만듭니다.
- **선행 태스크:** API-001, FE-002
- **핵심 가치 (VP):** [VP-1] 무한 페르소나와 일상 교감의 시작점

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
유저가 스스로 '영혼의 파트너'를 창조하는 몰입감 있는 경험을 제공해야 한다.

작업 지시:
1. `screens/PersonaSetupScreen.tsx`를 생성하라.
2. `react-hook-form`과 `@hookform/resolvers/zod`를 설치하라.
3. 미리 정의해둔 `PersonaSetupDTO`(API-001) 스키마를 가져와 폼 유효성 검사에 연결하라.
4. UI 구성:
   - "나만의 파트너 이름을 지어주세요" (TextInput)
   - "어떤 성격이었으면 좋겠나요?" (다정하게, 시크하게, 전문적으로 등 Dropdown 혹은 텍스트 입력창)
   - '설정 완료' 버튼
5. 폼 에러가 났을 때(예: 이름 누락) NativeWind 기반의 빨간색 안내 텍스트가 부드럽게 나타나도록 하라.
---

- [ ] 폼이 유효할 때만 제출 함수(onSubmit)가 정상 호출되는가?
