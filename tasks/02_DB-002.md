# [DB-002] PostgreSQL `vector` Extension 활성화 스크립트 실행

## 1. Task 개요
- **도메인:** Infra/DB
- **목표:** RAG(진화하는 지능) 시스템 구축을 위해 DB가 AI 임베딩 벡터를 이해하도록 확장 기능을 켭니다.
- **선행 태스크:** DB-001
- **핵심 가치 (VP):** [VP-2] 진화하는 지능 (Self-Evolution) 구현을 위한 물리적 기반

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
너는 Vesper AI의 데이터베이스 아키텍트다.
우리의 핵심 철학 중 하나는 **"유저와 AI의 대화를 모두 기억하고 진화하는 지능"**이다. 이를 위해 `pgvector` 확장이 필수적이다.

작업 지시:
1. `supabase/migrations/` 폴더에 타임스탬프 기반의 새 마이그레이션 파일(예: `20240418_enable_vector.sql`)을 생성하라.
2. 해당 파일 안에 PostgreSQL의 `vector` 확장을 활성화하는 SQL 명령어(`CREATE EXTENSION IF NOT EXISTS vector;`)를 작성하라.
3. 로컬 Supabase DB에 이 마이그레이션을 적용(`supabase db push` 또는 `supabase migration up`)하여 확장이 제대로 켜졌는지 확인하라.
4. 성공하면 나에게 "AI 지능을 담을 벡터 공간이 활성화되었습니다"라고 보고하라.
---
