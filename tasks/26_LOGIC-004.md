# [LOGIC-004] 대화 내역(User/Assistant) 벡터 DB 영구 저장

## 1. Task 개요
- **도메인:** Backend / Edge Function
- **목표:** 유저의 말과 AI의 대답 모두를 원문 텍스트 및 벡터(임베딩)와 함께 `heritage_logs` 테이블에 삽입(Insert)합니다.
- **선행 태스크:** DB-004, LOGIC-003
- **핵심 가치 (VP):** [VP-2] 나이를 먹듯 진화하기 위한 완벽한 기억력 이식 (양방향 RAG)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
과거 문서(v2.3)의 가장 큰 결함은 '유저의 말'만 기억했다는 것이다. 유저가 한 말과, 그에 대해 AI가 답변했던 내용 양쪽을 모두 기억해야 진정한 맥락 파악이 가능하다.

작업 지시:
1. `chat/index.ts` 내부 로직을 작성하라.
2. 현재 들어온 유저의 최신 메시지와 방금 변환한 임베딩 벡터를 Supabase DB의 `heritage_logs` 테이블에 INSERT 하라 (`role: 'user'`).
3. (중요) AI가 답변 생성을 완료하고 응답 스트리밍이 끝나는 시점(onFinish 훅 등 활용)에, **AI가 내뱉은 답변 역시 임베딩으로 변환하여 `heritage_logs` 테이블에 INSERT** 하라 (`role: 'assistant'`).
4. 백그라운드 쿼리로 처리하여 사용자 응답 속도에 방해(Latency)를 주지 마라.
