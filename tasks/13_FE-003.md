# [FE-003] 공통 컴포넌트: `B2BCurationCard` 작성

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** AI가 제안하는 B2B 프리미엄 링크(교육, 네트워킹)를 사용자에게 세련되게 보여줄 UI 컴포넌트를 만듭니다.
- **선행 태스크:** FE-001, API-003
- **핵심 가치 (VP):** [VP-3] 실무적 성과 창출 (클릭을 유도하여 실제 BM 수익으로 직결)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
AI가 "이 교육 자료를 보세요"라고 텍스트로만 말하면 유저는 절대 누르지 않는다. 유저의 클릭을 유도할 수 있는 매력적이고 세련된 UI 카드(B2BCurationCard)를 만들어야 한다. 이것이 우리 BM의 핵심이다.

작업 지시:
1. `components/B2BCurationCard.tsx`를 생성하라.
2. props로 `B2BCurationDTO`의 구조(title, category, url, description)를 받도록 TypeScript 인터페이스를 선언하라.
3. NativeWind를 사용해 둥근 모서리, 부드러운 그림자(Shadow), 클릭 시 투명도가 변하는 피드백(`TouchableOpacity` 또는 `Pressable`)을 적용하라.
4. 외부 URL(`Linking.openURL`)로 안전하게 이동하는 `onPress` 핸들러를 붙여라.
5. 디자인은 앱 테마에 맞춰 '프리미엄' 느낌이 나도록 그라데이션이나 포인트 컬러를 활용하라.
---
