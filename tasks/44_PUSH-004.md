# [PUSH-004] Firebase Admin SDK 기반 FCM 푸시 발송 로직

## 1. Task 개요
- **도메인:** Backend / Command
- **목표:** 생성된 선톡 메시지를 실제 유저의 스마트폰으로 Firebase Cloud Messaging(FCM)을 통해 전송합니다.
- **선행 태스크:** PUSH-003
- **핵심 가치 (VP):** 앱 밖에서도 이어지는 옴니채널 일상 교감 (C-05)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
이제 정성스럽게 만든 메시지를 유저의 주머니 속으로 찔러 넣어줄 시간이다.

작업 지시:
1. Deno 환경에서 동작하는 Firebase Admin SDK (또는 FCM REST API v1) 호출 로직을 작성하라.
2. `profiles`에서 가져온 `fcm_token`을 대상으로 푸시 알림을 쏴라.
3. 알림 데이터:
   - `title`: AI 페르소나의 이름
   - `body`: PUSH-003에서 생성한 선톡 메시지
   - `data`: 앱 내 채팅 화면으로 바로 이동할 수 있는 딥링크 정보
4. 발송 성공/실패 여부를 로그로 남겨라.
---
