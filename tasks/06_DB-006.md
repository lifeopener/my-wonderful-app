# [DB-006] `b2b_curations` 테이블 및 시드 데이터 카탈로그 생성

## 1. Task 개요
- **도메인:** DB Schema
- **목표:** B2B 수익화 모델의 기반이 되는 프리미엄 실무 자료, 네트워킹 모임 등의 큐레이션 링크를 담을 카탈로그 테이블을 만듭니다.
- **선행 태스크:** DB-001
- **핵심 가치 (VP):** [VP-3] 실무적 성과 창출 (단순 위로가 아닌 솔루션 제공)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
우리의 AI는 단순한 대화형 챗봇이 아니라, 유저에게 **"즉시 실행 가능한 실무적 솔루션과 수익 모델(BM)"**을 제공하는 비즈니스 파트너다. 이를 위해 프리미엄 링크를 담을 DB가 필요하다.

작업 지시:
1. 새 마이그레이션 파일을 생성하고 `public.b2b_curations` 테이블 DDL을 작성하라.
2. 다음 컬럼을 포함하라:
   - `id`: UUID (PK)
   - `category`: TEXT ('education', 'networking', 'investment' 등)
   - `title`: TEXT (표시할 제목)
   - `url`: TEXT (연결할 프리미엄 링크)
   - `description`: TEXT (간단한 설명)
   - `created_at`: TIMESTAMP
3. 생성된 테이블에 AI가 참조할 수 있도록 카테고리별 시드 데이터(Seed Data) 3~4개를 INSERT 하는 스크립트를 파일 하단에 추가하라.
4. 로컬 DB에 마이그레이션을 적용하여 시드 데이터가 들어갔는지 확인하라.
---
