# [AI-001] `getRealTimeNews` (Tavily 연동) Tool Calling 로직

## 1. Task 개요
- **도메인:** Web Search
- **목표:** AI가 최신 시장 상황을 모를 때 인터넷 검색을 수행할 수 있도록 Tavily Search API를 Tool Calling 형태로 연동합니다.
- **선행 태스크:** None (Tavily API Key 필요)
- **핵심 가치 (VP):** [VP-3] 실무적 성과 창출 (최신 정보를 바탕으로 한 솔루션)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
우리 AI는 과거에 머물러 있으면 안 된다. 모르는 게 있으면 인터넷을 검색해서 유저에게 알려줘야 한다.

작업 지시:
1. `supabase/functions/chat/tools/` 디렉토리에 `search.ts`를 생성하라.
2. Vercel AI SDK의 `tool` 함수를 사용하여 `getRealTimeNews` 툴을 정의하라.
3. `parameters`는 Zod를 사용하여 `query` 문자열을 받도록 설계하라.
4. `execute` 함수 내에서 Tavily API를 호출하여 검색 결과를 텍스트로 가공해 반환하라.
5. (중요) 성능을 위해 타임아웃을 1500ms로 설정하고, `Deno.env.get('TAVILY_API_KEY')`를 사용하라.
---

## 3. 검증 기준 (QA)
- [ ] 툴 호출 시 Tavily API에 검색 쿼리가 정상적으로 전달되는가?
- [ ] 타임아웃 발생 시 전체 응답이 죽지 않고 에러를 우아하게 처리하는가? (M-01)
