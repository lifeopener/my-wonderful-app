# [FE-012] Tool Calling 응답 커스텀 렌더링 (B2BCurationCard 연동)

## 1. Task 개요
- **도메인:** Frontend / UI
- **목표:** AI가 텍스트 대신 `B2BCurationDTO` 형식의 데이터를 내려줬을 때, 이를 인식하고 예쁜 카드 형태(B2BCurationCard)로 렌더링합니다.
- **선행 태스크:** FE-003, FE-004, FE-010
- **핵심 가치 (VP):** [VP-3] 실무적 성과 창출 (클릭을 유도하여 비즈니스 모델 완성)

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
가장 중요한 비즈니스 로직이다. AI가 Vercel AI SDK의 Tool Calling을 통해 "이 링크를 띄워주세요"라고 JSON 데이터를 보내면, 프론트엔드가 이를 낚아채서 멋진 카드로 그려내야 한다.

작업 지시:
1. `ChatMessageBubble.tsx` 내부 렌더링 로직을 수정하라.
2. 메시지 객체(message.toolInvocations) 안에 `toolName === 'getB2BCuration'` 등의 데이터가 들어오면, 일반 마크다운 텍스트 렌더링을 멈춰라.
3. 대신 FE-003에서 만들어둔 `B2BCurationCard` 컴포넌트를 호출하고, 인자로 넘어온 데이터(title, url 등)를 넘겨주어 화면에 카드가 뜨도록 하라.
4. 카드가 여러 개 올 경우 가로 스크롤(Horizontal Scroll)로 예쁘게 나열하라.
---

- [ ] 카드 클릭 시 외부 URL로 잘 이동하는가?
