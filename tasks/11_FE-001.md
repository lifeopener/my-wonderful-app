# [FE-001] Expo + NativeWind 초기 프로젝트 셋업

## 1. Task 개요
- **도메인:** Frontend / Infra
- **목표:** 모바일 앱(iOS/Android)의 뼈대가 될 Expo(React Native) 프로젝트를 생성하고, Tailwind CSS 기반의 NativeWind를 연동합니다.
- **선행 태스크:** 없음
- **핵심 가치 (VP):** 유려하고 부드러운 앱 사용성을 제공하여 유저의 '초밀착 인지 동행'을 시각적으로 지원합니다.

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
너는 이제 Vesper AI의 프론트엔드 수석 개발자다. 
우리의 목표는 유저가 매일 접속하고 싶은, 빠르고 아름다운 모바일 앱을 구축하는 것이다.

작업 지시:
1. `npx create-expo-app@latest vesper-app` 명령어를 통해 최신 Expo 템플릿(TypeScript)으로 프로젝트를 생성하라.
2. `nativewind` 및 `tailwindcss` 패키지를 설치하고 연동하라.
3. `tailwind.config.js`를 셋업하고, 앱의 메인 테마(Vesper를 상징하는 저녁별, 다크 모드 기반의 우아한 색상 등) 변수를 추가하라.
4. `App.tsx`를 수정해 화면 정중앙에 Tailwind 클래스로 스타일링된 "Vesper AI Companion OS" 텍스트가 뜨는지 확인하라.
5. `npm run ios` 또는 `npm run android` 스크립트를 통해 로컬 빌드가 정상 작동하는지 점검 후 완료를 보고하라.
