# [LOGIC-001] Edge Function `chat` 생성 및 기본 세팅

## 1. Task 개요
- **도메인:** Backend / Edge Function
- **목표:** AI 통신을 담당할 서버리스 함수인 `chat` 함수를 Supabase에 생성합니다.
- **선행 태스크:** DB-001
- **핵심 가치 (VP):** 고정비 0원 아키텍처의 핵심 심장부

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
우리는 값비싼 Node.js나 Spring 서버를 유지할 돈이 없다. 모든 백엔드 로직은 Supabase의 Deno 기반 Edge Function 안에서 처리되어야 한다.

작업 지시:
1. `supabase functions new chat` 명령어로 새로운 엣지 펑션을 생성하라.
2. `supabase/functions/chat/index.ts` 파일을 열어라.
3. CORS 헤더 처리를 완벽하게 추가하여, 모바일 앱이나 웹 프론트엔드에서 호출할 때 CORS 에러가 나지 않도록 방어하라.
4. HTTP POST 요청에 대해 "Hello Vesper"를 반환하는 뼈대 코드를 작성하고 `supabase functions serve chat`으로 로컬 테스트가 가능한지 점검하라.
