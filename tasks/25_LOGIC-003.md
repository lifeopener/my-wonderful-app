# [LOGIC-003] OpenAI Embeddings API 연동 (텍스트 -> 벡터 변환)

## 1. Task 개요
- **도메인:** Backend / Edge Function
- **목표:** 유저가 보낸 메시지를 1536차원의 수학적 벡터(숫자 배열)로 변환합니다.
- **선행 태스크:** LOGIC-001
- **핵심 가치 (VP):** [VP-2] 진화하는 지능을 위한 의미론적 데이터 변환의 시작

## 2. Vibe Coding 프롬프트 (AI 입력용)

---
**[Prompt]**
단순한 텍스트 매칭(키워드 검색)으로는 유저의 감정을 이해할 수 없다. 유저의 문장을 OpenAI의 두뇌를 빌려 수학적 좌표(벡터)로 변환해야 한다.

작업 지시:
1. `chat/index.ts`에 OpenAI SDK를 임포트하라.
2. 유저가 보낸 최신 메시지(문자열)를 가져와라.
3. OpenAI의 `text-embedding-3-small` 모델(또는 `text-embedding-ada-002`)을 호출하여 텍스트를 1536차원의 벡터 배열로 변환하는 함수 `generateEmbedding(text)`를 작성하라.
4. 환경변수(`OPENAI_API_KEY`)를 안전하게 불러오는지 점검하라.
---

- [ ] 반환된 결과물이 1536개의 숫자로 이루어진 배열(Array) 형태인가?
